# Interactions

Interactions is our user interface where you can ask questions about your own data. It's available under the project's submenu:

![image](https://github.com/StubbornDeer/enum-docs-nextra/assets/91156314/4bfedfb4-a4f8-412f-94e0-949a2732689d)


To start a new interaction, click the "Chat" button on the top, then ask a question in the message box below. 

<Callout>
You may click the button with a plain icon to send your message, or just press the Enter button on your keyboard.
</Callout>

Here is the interaction interface:

![image](https://github.com/StubbornDeer/enum-docs-nextra/assets/91156314/7f18d97a-e3d3-411c-bd8c-aafcdd2f4759)

Every response has a label indicating a current ChatGPT model as well as the number of spent tokens.

For AI tuning, use the custom prompt that is available in the project settings.

## Showing sources
To show sources on the interaction page, please set up the corresponding option in the project's settings (*Show the sources in Interactions*):

![image](https://github.com/user-attachments/assets/71a1705a-7072-4b6a-bc48-9e1a7215af8c)


The amount and type of sources (main/additional) are set up according to the project's source settings.

<Callout>
Please note, that these settings may be different from the chatbot's ones.
</Callout>

## Response in JSON format
